# docs/

This directory contains the public documentation for the VinMin project under
TLTE C.I.C. Only non-sensitive, public-facing information is stored here.

## Structure

- `public/` — general purpose documentation  
- `guides/` — contributor and developer guides  
- `api/` — (optional) public API references  
- `index.md` — entry point for documentation  

Internal or sensitive documentation is not included in this repository.
