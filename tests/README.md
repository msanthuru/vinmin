# tests/

This directory contains public-safe tests for the VinMin project under
TLTE C.I.C. Tests placed here should cover open-source modules, utilities,
and components that are intended for community use.

Only public, non-sensitive code should be tested in this repository.

## Allowed tests:
- unit tests for public modules  
- integration tests for safe components  
- test helpers and fixtures  
- example tests for contributors  

## Not allowed:
- tests involving internal TLTE systems  
- identity, authentication, or verification logic  
- region-sensitive processes  
- private infrastructure functions  

Internal and security-sensitive testing remains private within TLTE C.I.C.
secure environments.
