# src/

This directory contains the public-safe source code for the VinMin project under
TLTE C.I.C. Only non-sensitive modules, utilities, and components will appear
here.

## Structure

- `core/` — foundational logic and shared functionality  
- `utils/` — helper functions, formatters, validators  
- `services/` — external-facing service handlers  
- `config/` — non-secret configuration files  
- `public-modules/` — components intended for public release  

Internal, private, or security-sensitive systems will not be present in this
repository.
